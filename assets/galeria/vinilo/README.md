# Carpeta de Imágenes - Vinilos y DVD Pintados

Coloca aquí las fotos de ejemplo de discos de vinilo y DVD pintados.

## Instrucciones:
- Nombra las imágenes como: `1.jpg`, `2.jpg`, `3.jpg`, etc.
- Formatos aceptados: JPG, PNG, WebP
- Tamaño recomendado: 1200x900px o mayor
- Las imágenes se mostrarán automáticamente en la galería del sitio

## Sugerencias:
- Usa fotos de buena calidad con buena iluminación
- Muestra diferentes diseños y estilos
- Incluye fotos de diferentes tamaños (12, 18, 25, 30 cm)
